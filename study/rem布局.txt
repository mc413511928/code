rem布局
1可以让宽高等比例缩放自适应，felx和流式布局主要针对宽带自适应来布局的。
2rem单位，是一个相对单位
	em是相对于父元素的字体大小来说的，
	rem是相对于html元素字体大小的。
3rem的优点
	1可以通过修改html里面的文字大小来改变页面中元素的大小，可以整体控制，
4媒体查询
	1，针对不同屏幕尺寸设置不同的样式。
	2,@media开头 关键字and  not only
	3语法规范，all  print scree,
	4,关键字，and相当于且的意思
		not相当于非 的意思，可以省略
		only指定某个特定的媒体类型。于可以省略。
	5媒体特性
		1用小括号包含，
		2媒体查询可以根据不同的屏幕尺寸来改变不同的样式，
		3最大值最小值都是包含等于的。
		4媒体查询一般按照从小到大的顺序来
		5screen和and必须带上，不能省略，
		6我们的数字后面必须跟单位，这个px不能省略的。
		max小于等于
		min大于等于
5引入资源
		一个建议：我们媒体查询最好的方法是从小到大。
		引入资源就是针对不同屏幕尺寸设置不同的样式。
6less 		1css没有计算能力，不方便维护，不利于重复，
		2lesss 是一门css扩展语言，为css加入程序语言的特性，
		语法@变量名：值；
		见名知意，变量名是区分大小写，不能以数字开头，不能有特殊符号，
6less嵌套。
		1子元素的样式直接写到父元素里面去,(儿子住到爸爸家去）
		2伪类选择器：用&如果有伪类交集选择器，我们内选择器前面要加&,
7less运算
		1我们的乘号是*除号是/
		2在我们运算符的左右两侧必须敲一个空格隔开，
		3两个数参与运算，如果只有有一个数有单位，结果就以这个单位为单位，
		4如果两个数参与运算；两个数都有单位，并且不一样，
		5颜色也可以运算的，
rem适配方案
		1  技术方案1：rem+媒体查询+less技术
		2屏幕宽度/划分分数=html字体大小，
		①最后的公式：页面元素的rem值 =  页面元素值（px） /  （屏幕宽度  /  划分的份数）

		②屏幕宽度/划分的份数就是 htmlfont-size 的大小

		③或者：页面元素的rem值 =  页面元素值（px） /  html font-size 字体大小


		2因为我们pc端也可以打开我们苏宁移动端首页，我们默认html字体大小为 50px，注意这句话写到最上面

rem 第二种适配方案；
	需要修改cssroot的默认大小手动改成我们需要的字体大小快捷键crl+,
	html的字体大小设计稿大小/分数
如果我们的屏幕超过设计稿屏幕，我们按照设计稿来走，不会让页面超过设计稿尺寸，
解决方案
	@media screen and (min-width:750px) {
    html {
        font-size: 75px !important;
    }
}    
	这里的750是设计稿尺寸，
	75px是设计稿的html的字体大小，
